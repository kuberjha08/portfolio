# 🧰 Getting Started

### 1. Install Requirements
Make sure **Git** and **NodeJS** are installed.

### 2. Clone Repository

```bash
git clone https://github.com/kuber/portfolio.git
cd portfolio
```

### 3. Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

### 4. Run Development Server

```bash
npm run dev
```

or

```bash
yarn dev
```

### 5. Open in Browser

```
http://localhost:3000
```
